# ioput.h-C-
ioput.h - A Lightweight C++ I/O Library  A minimal, header-only C++ I/O library using system calls (read/write). Zero dependencies, type-safe, perfect for learning and embedded systems.  Features: Simple input()/output() API • Formatting (setlg/setlgc/setdep) • Terminal controls (nl/ll/clear) • UTF-8 with Chinese support • High performance
